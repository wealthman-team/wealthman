## Развертывание проекта

composer install

npm install

php artisan migrate

php artisan db:seed

php artisan key:generate

npm run development

#### Заметки
Перед выполнение команды npm install, возможно потребуется установка npm install --global windows-build-tools
для правильной работы webfonts-loader
